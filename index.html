How to Avoid Bans Using Stand

Avoid Account Sharing: Stand has a strict policy against account sharing. The automated system (B4) is highly sensitive to this activity. If detected, it results in an immediate permanent suspension without the possibility of review. Never share your account ID with anyone.

Regenerate License Responsibly: You can only regenerate your activation key twice a day. Excessive regeneration (D1) of your activation license can lead to a permanent suspension. Make sure to regenerate your license only when necessary and avoid any actions that may trigger the automated system. Sharing your account exacerbates this violation, leading to a swift suspension.

Troubleshooting Guide for Stand Users

Apply Default State:

  • Focus Link: Stand > Apply Default State
  • Command Box Link: U > default > Enter
  • If you encounter an issue, first apply the default state through the provided link.
  • If the issue stops persisting after enabling default, it's likely a function you've saved to the profile.
  • If the issue continues with default enabled, it's likely not Stand related.

Disable Mods:

  • Disable Lua/ASI mods and OpenIV mods to troubleshoot.
  • Check if the issue is resolved after disabling these mods.
  • If the issue persists even after disabling mods, it's likely not caused by external modifications.

Seeking Help:

  • Ifter trying default state and disabling external mods your issue still is not resolved, seek help in #support-and-help.
  • Explain that you've tried default and removed external mods to no avail.
