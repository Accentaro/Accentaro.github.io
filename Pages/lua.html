Lua Scripts Installation Guide

Lua Scripts are files that unlock additional functionality within Stand. Other mod menus also use Lua as a scripting language, but their scripts are generally not compatible with Stand and vice-versa.

Obtain the script file:

  • Scripts can be downloaded from anywhere, but usually are found either in the built-in script repository (Stand > Lua Scripts > Repository) or in this Discord's lua-scripts channel.
  • The main script file is generally a *.lua or *.pluto file extension. Scripts are text files that can be edited in a text editor, but should NOT have the *.txt extension.
  • If downloading a project from GitHub.com you can download a zip file of the entire project by clicking the big green Code button and selecting Download Zip. GitHub adds a containing folder for the project inside the zip file, something like project-name-main. You want to copy the content within that folder, but not the folder itself.

Install your script file into the Stand/Lua Scripts folder:

  • The built-in repo is the easiest installation option (just check the box to install) but not all scripts are in the repo.
  • You can find your Stand folder by clicking the Open Stand Folder button in the Stand Launchpad, or in the menu itself at Stand>Open Folder.

Run the script:

  • Browse your installed scripts at Stand>Lua Scripts.
  • Select a script and then Start Script to run it and expose any additional menu items it may have.
  • You don't need to restart anything to view newly installed scripts, simply re-enter the Lua Scripts menu to refresh it.
  • Some scripts have auto-updaters or other features that can be enabled if you enable internet access for the script. You should only do this for scripts you trust.
