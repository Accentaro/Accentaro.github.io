Taking Host Guide

  1. Goto Online > Spoofing > Host Token Spoofing
  2. Enable it
  3. Set the Preset to Aggressive - Online > Spoofing > Host Token Spoofing > Presets > Aggressive
  4. For Ultimate Users only turn on Kick Host When Joining As Next In Queue - Online > Spoofing > Host Token Spoofing > Kick Host When Joining As Next In Queue. For anyone without Stand Ultimate you will have to manually kick/crash the host when joining a lobby.
  5. Rejoin the session or find a new lobby

Focus Link: Online > Session > Rejoin Session

Command Box Link: U > rejoin > Enter
